‚öôÔ∏è Theoretical Background

This project is based on sequential circuits where output depends on current inputs and previous states:

  • Logic Gates: AND, OR, NOT gates create logical expressions
  • Flip-Flops: D Flip-Flops store and change states with each clock pulse
  • Counters: Generate timing for light changes
  • Boolean Expressions: Define logic relationships between inputs and outputs
  • Karnaugh Maps: Simplify Boolean expressions to minimize circuit complexity

üîå Circuit Design

The circuit consists of three D flip-flops representing three traffic light states:

  • State 00: Green Light ON
  • State 01: Yellow Light ON
  • State 10: Red Light ON

Each state transition occurs on the rising edge of the clock pulse. The logic circuit uses AND/OR gates to determine the next state.
